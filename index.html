Problema 1
Resultado: a₂₀ = 64
Desenvolvimento:
Em uma PA, o termo geral é
aₙ = a₁ + (n − 1) · r
a₂₀ = 7 + (20 − 1) · 3
a₂₀ = 7 + 19 · 3
a₂₀ = 7 + 57
a₂₀ = 64
Ideia-chave
Aplicar o termo geral da PA para achar o 20º termo.
Problema 2
Resultado: a₁ = 2 e r = 4
Desenvolvimento:
Numa progressão aritmética, o termo geral é
aₙ = a₁ + (n − 1)r
Para n = 5:   a₅ = a₁ + 4r = 18
Para n = 13:  a₁₃ = a₁ + 12r = 50
Subtraindo as equações:
(a₁ + 12r) − (a₁ + 4r) = 50 − 18
8r = 32  ⟹  r = 4
Voltando em a₅ = a₁ + 4r:
a₁ + 4 · 4 = 18  ⟹  a₁ = 2
Ideia-chave
Comparar dois termos da PA pelo termo geral; ao subtrair as equações, a₁ se elimina e a razão surge pela diferença entre os índices (13 − 5 = 8). Com r encontrado, determina-se a₁.
